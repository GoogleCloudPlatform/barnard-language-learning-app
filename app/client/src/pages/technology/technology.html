The technology

Built with Google Translate and Cloud Vision technology, Woolaroo uses advanced machine learning to understand the objects in your photos and translates them, in real time.

The resulting translation is based on the key object in the image. If multiple objects are detected, you can scroll through and select the correct translation.

Woolaroo relies on the latest AI technology so it may not get it right every time, but we're working hard to improve it. You can help us by using the Translation Feedback feature to log any translations that don't seem right.
