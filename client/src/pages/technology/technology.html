The technology

Built with Google Translate and Cloud Vision technology, Project Woolaroo uses advanced Machine Learning to understand the objects in your photos and translate them, in real time.

It serves up the most likely indigenous translation based on the key object in an image. If multiple objects are detected, you can scroll through and select the correct translation.

Project Woolaroo relies on the latest AI technology so it may not get it right every time, but we're working hard to improve it. You can help us by using the Translation Feedback feature to log any translations that don't seem right.
